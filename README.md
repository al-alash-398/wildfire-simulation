Симуляция распространения лесного пожара с помощью клеточного автомата

# Идея проекта
Симуляция лесного пожара на 2D сетке. Каждая клетка может быть пустошью, деревом, огнём или пеплом.
Физические факторы: ветер - задаёт направление распространения огня, разные деревья - разные характеристики, влажность - снижает вероятность возгорания.
Визуализация: цветовая карта, анимация распространения и графики статистики.

# Библиотеки
Из курса: numpy, matplotlib 
Не из курса: numba - ускорение, pydantic - валидация 

# Фичи к итерации 1
1. Клеточный автомат (распространение огня на соседей)
2. Анимация распространения
3. Цветовая карта с легендой

# Фичи к итерации 2
1. Ветер (направленное распространение)
2. Влажность
3. Типы деревьев (сосна, берёза, дуб)
4. Статистика и графики
